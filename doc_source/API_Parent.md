# Parent<a name="API_Parent"></a>

A parent label for a label\. A label can have 0, 1, or more parents\. 

## Contents<a name="API_Parent_Contents"></a>

 ** Name **   <a name="rekognition-Type-Parent-Name"></a>
The name of the parent label\.  
Type: String  
Required: No
